| [Définir des interfaces interopérables](interoperability.md) | Pour composer plusieurs algorithmes |
| [Écrire des tests de conformité](compliance-tests.md) | Pour valider la correction réglementaire |
| [Automatiser la publication (CI/CD)](cicd.md) | Pour industrialiser les releases |
